# Project Commands

This is a template. Replace it with your project's actual commands by creating
a `project-commands.md` file in your project config directory.

The agent uses these commands during implementation (Phase 5), code review
(Phase 6), and final polish (Phase 8). Providing accurate commands here
prevents the agent from guessing and potentially thrashing on basic operations.

Only include commands that apply to your project. Remove or omit sections that
are not relevant.

## Build

```
# command to build the project
```

## Lint

```
# command to run linting
```

## Format

```
# command to run code formatting
```

## Test (all)

```
# command to run all tests
```

## Test (single file)

```
# command to run a single test file
# use FILE as a placeholder for the file path
```

## Type Check

```
# command to run type checking (if applicable)
```

## Dev Server

```
# command to start the dev server (if applicable)
```
